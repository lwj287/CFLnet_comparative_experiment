"a_UIEB", "a_LSUI", "a_EUVP" are different dataset paths. a_compare includes multiple comparison methods.

For UCM, IBLA, UDCP, and ULAP, modify the dataset name and run "main.py" to obtain enhanced images for different datasets, which are stored in "enhs_[dataset]". Run "evaluate_[dataset].py" to obtain comparative images, which are stored in "vals_[dataset]".

For DNnet, LiteEnhanceNet, run "test_[dataset].py" to obtain enhanced images from different datasets, store them in "enhs_[dataset]", run "evaluate_[dataset].py" to obtain comparative images, store them in "vals_[dataset]".
